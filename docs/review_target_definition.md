# 검토대상객체 정의

## 목적

본 문서는 기능별 검토 로직에서 사용하는 `검토대상객체`의 의미를 통일하기 위한 기준 문서이다.
새 기능을 정의하거나 검토 기준을 작성할 때, 별도 예외가 명시되지 않는 한 본 문서의 정의를 우선 적용한다.

---

## 표준 정의

`검토대상객체`란 Revit 모델 내 객체 중 실제 시공 또는 설치되는 물리적 모델 객체로서, 모델 안에서 형상 또는 부피를 가지며 건축물 또는 시설물의 구성요소로 취급되는 객체를 말한다.

검토 기준에 `전 객체 대상`이라고 명시된 경우, 위 정의에 해당하는 모든 물리적 모델 객체를 대상으로 한다.

---

## 판단 기준

다음 조건을 만족하는 경우 `검토대상객체`로 본다.

1. 모델 내에서 실제 형상 또는 부피를 가진다.
2. 실제 시공 또는 설치의 대상이다.
3. 공간을 점유하거나 물리적 경계를 형성한다.
4. 단순 참조, 표기, 해석, 주석, 보조 목적의 객체가 아니다.

## 적용 원칙

1. `전 객체 대상`은 항상 `실제 시공되는 물리적 모델 객체 전체`를 의미한다.
2. 비물리적 참조 객체는 기본적으로 검토대상에서 제외한다.
3. 특정 기능에서 예외가 필요한 경우, 해당 기능 명세에 예외 대상을 명시적으로 추가한다.
4. 예외가 명시되지 않았다면 본 문서의 정의를 기본값으로 적용한다.
5. 특히 `System`, `Material`, `Project Information`, `Sun Path`, `Pipe Segment`, `Primary Contour`, `Legend Component`는 검토대상에서 제외한다.

## 기능 명세용 템플릿

### 기본 템플릿

```text
검토대상: 전 객체 대상

정의:
실제 시공 또는 설치되는 물리적 모델 객체 전체를 의미한다.
센터라인, 주석, 태그, 치수, 그리드, 레벨 등 비물리적 참조 객체는 제외한다.
```

### 상세 템플릿

```text
검토대상객체는 Revit 모델 내 객체 중 실제 시공 또는 설치되는 물리적 모델 객체를 의미한다.
본 기능에서 "전 객체 대상"은 형상 또는 부피를 가지며 실제로 모델링되는 모든 시공 객체를 대상으로 한다.
센터라인, 주석, 태그, 치수, 심볼, 그리드, 레벨, 참조평면 등 비물리적 참조 객체는 검토대상에서 제외한다.
```

### 예외 포함 템플릿

```text
검토대상: 전 객체 대상
예외: [여기에 제외 또는 추가 대상을 명시]

기본적으로 실제 시공 또는 설치되는 물리적 모델 객체 전체를 대상으로 한다.
단, 본 기능에서는 [예외 내용]을 추가로 적용한다.
```

---

## 구현 메모

- 기능 구현 시 `검토대상 지정`이 별도 입력값으로 존재하면, 기본 옵션은 본 문서의 `전 객체 대상` 정의를 따라야 한다.
- 카테고리 필터를 구현할 때는 `실제 시공 여부`와 `물리적 모델 여부`를 우선 판단 기준으로 사용한다.
- 선형 참조 요소, 주석 요소, 문서화 요소는 기본적으로 제외 목록에 둔다.
