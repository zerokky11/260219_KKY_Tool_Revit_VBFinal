KKY Tool Update Server

This folder is intended for GitHub Pages update hosting.

Upload these files to the repository root:
  • index.html
  • .nojekyll
  • latest.json
  • Your installer file, for example KKY_Tool_Revit_v2.03.exe
Expected update feed URL:

https://your-github-username.github.io/your-repo-name/latest.json
